## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/AyresIV/ayresiv.com.git
cd ayresiv.com

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5175`
